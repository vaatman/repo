Install Zabbix server on Raspberru pi400
========================================

.prepare - create OS SD-CARD
  .get 128GB SD-CARD
  .put in laptop
  .install rpi-imager on debian13
    $ sudo apt update
    $ sudo apt install snapd
    $ sudo snap install snapd
    $ sudo snap install rpi-imager
    .install in you OWN $HOME
      $ snap install snapd
      $ snap run rpi-imager
      :write to th SD-CARD as you like
        - Device: Raspberry Pi4
        - OS: Raspberry OS (64BIT)
        - Storage: Internal SD card reader
