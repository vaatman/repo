Install Image Builder on RHEL9
==============================

. prepare a RHEL9 VM
  - qemu/libvirt
  - virt-manager
  - rhel9.7 standard server
  :source https://www.redhat.com/en/blog/linux-golden-homelab-rhel
  :also read:
    https://medium.com/@jaine.mayank/rhel-imagebuilder-e892a7ebea17
    https://docs.redhat.com/en/documentation/red_hat_enterprise_linux/8/html/composing_a_customized_rhel_system_image/creating-system-images-with-composer-web-console-interface_composing-a-customized-rhel-system-image

. install Image Builder
  # dnf update
  # dnf install -y cockpit-composer.noarch composer-cli

. start enable the socket
  # systemctl enable --now osbuild-composer.socket
    .extra: start and enable cockpit
    # systemctl enable --now cockpit

. browse to webUI
  https://rhel9-1:9090/composer
  > click 'Limited access' when visible up-right
    to gain 'Administrative access'
  :should show 'Create blueprint' button

  > wait for image build to start
   "rhel-base: Image creation has been added to the queue."

. CLI stuff
  . list blueprints
    # composer-cli blueprints list
  . show the blueprint created above
    # composer-cli blueprints show rhel-base | less
  . change local account name in blueprint
    # composer-cli blueprints show rhel-base > rhel-base.blueprint
    # vi rhel-base.blueprint
      admin -> ansible
    # composer-cli blueprints push rhel-base.blueprint 

  . save a blueprint [as created in eg. cockpit]
    # composer-cli blueprints save rhel-base-001
  . edit with vi, see rhel-base-001.toml in the repo
  . push the blueprint
    # composer-cli blueprints push rhel-base-001
  . build an image
    # composer-cli compose start rhel-base-001 qcow2 

  . status
    # composer-cli compose status
  . logs from a job
    # composer-cli compose log 9d11d30f-e5c5-4a27-ae0b-8997fc872b85
    The compose result is empty.
  . worker logs
    # journalctl -u osbuild-composer -u osbuild-worker -n 200 --no-pager

  . test a minimal image
    # vi minimal.toml
name = "minimal"
description = "minimal qcow2"
version = "0.0.1"
    # composer-cli blueprints push minimal.toml
    # composer-cli compose start minimal qcow2
